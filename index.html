Data Science Projects

Air Quality Forcasting

Used: Python, Machine Learning (TensorFlow), and Jupyter Notebooks


Datasets from Environment Canada were used to train three different machine learning models to predict hourly concentrations of air pollution in six major Canadian cities. This dataset spanned five years and contained 52,584 data points for each of the six cities. Data for hourly temperature, dew point, air pressure, wind speed, ozone concentration, nitrogen oxide concentration, and PM2.5 concentration was used for each city to train these models.

Single Cell RNA Sequencing

Used: R, MATLAB, Machine Learning (MATLAB Classification Learner App) and SPSS


Developed a data pipeline in R so that a research lab could process, visualize, and analyze single cell RNA-sequencing datasets. This involved working with a large dataset stored in a computer cluster and applying a few machine learning models. The overall goal was to identify changes occuring between specific cell types in healthy vs diseased specimen.
